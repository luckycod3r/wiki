Через главную функцию можно легко получать элементы


console.log(A_(".aqua-virtual-console"))

Кастомные классы

Позволяет создавать свои классы на стороне JS без использования CSS-файла 

 
let myClass = {
    backgroundColor: "#f00",
    color: "#fff"
}
let result = new Class(myClass);
console.log(result.name) // выведет название класса
new Listener(".unsetClass","click",(event)=>{
    result.unset(A_(".genericClass"));
});
result.applyTo(A_(".genericClass"));

Диалоги

Очень удобная возможность библиотеки. При вызове создаёт диалоговое окно с вариантом ответа Да/Нет new Dialog(object) 


let elem = A_(".dialogRunner"); // Ищем кнопку
// Добавляем слушатель событий на событие - клик
new Listener(elem,"click",(event)=>{
    // Конфигурируем диалог
       let dialog = {
           title: "Заголовок",
           message: "Сообщение",
           onYes: function(){
               alert(`Вы нажали ДА`)
           },
           onDecline : function(){
               alert(`Вы нажали НЕТ`)
           }
       }
       // создаём объект диалога
       let dialogObject = new Dialog(dialog)
       dialogObject.open() // открываем диалоговое окно
})
Run!
